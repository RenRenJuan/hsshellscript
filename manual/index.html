HsShellScript User Manual

This is the user manual for the HsShellScript Haskell shell scripting library, version 3.3.2. It has been released 2014-08-06. The API documentation is in a separate document.

HsShellScript is a library which makes things easy to program in Haskell, which are typically done by shell scripts on Unix-like systems. You can use Haskell for writing your shell scripts.

So your Haskell scripts can grow to become real programs. It's all ready.
