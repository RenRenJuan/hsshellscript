Requirements

Requirements for using and for building

  • The Glasgow Haskell Compiler. It works with GHC-7.8.3, and should also work with versions 7.8.0 upwards. Because of incompatible changes in GHC, it does not work with GHC-7.6 and older.

  • A Unix like system. HsShellScript is being developed and tested on Linux.

Requirements only for building

  • GNU make

  • The interface generator C2HS

  • Haddock for the API documentation

GHC and Haddock are included in the The Haskell Platform.
